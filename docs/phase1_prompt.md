# Phase 1: Build (AI Allowed)

Build a premium React developer dashboard with:

- Vite + React + Tailwind
- Clean Apple-style UI
- Responsive layout

Features:
1. Dashboard
   - GitHub summary
   - Weather widget

2. GitHub Page
   - Search user
   - Show repos
   - Filter + sort repos

3. Tasks Page
   - Add / delete / complete tasks
   - Persist in localStorage

4. 404 Page

Rules:
- Use React Router
- Use reusable components
- No inline API calls in components
- Keep UI minimal and elegant
